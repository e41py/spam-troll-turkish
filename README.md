# 🚀 Otomatik Mesaj Spam Botu

Bu proje, **CustomTkinter** kullanarak modern bir **otomatik mesaj spam botu** oluşturur.\
Şık bir\*\* arayüz,\*\* **açılış/kapanış animasyonu** ve **otomatik fullscreen** özellikleri ile geliştirilmiştir.
